# git命令
git init 初始化当前目录  
git status 查看当前文件状态
git branch 查看分支 *表示当前分支  
git checkout branch 切换分支branch  
git push 提交到远程代码
git pull 从远程拉取代码
git checkout -b dev 创建并切换到dev分支
合并分支代码  
git add  
git commit -m '提交描述'  
git checkout main 切换到需要合并到的分支  
git merge dev （dev是从dev合并过来的分支）  
git branch -d dev(删掉dev分支)如果需要
